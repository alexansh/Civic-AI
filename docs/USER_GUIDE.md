## Getting Started

### Registration

1. Visit the CIVIC AI website
2. Click "Sign Up"
3. Choose your account type:
   - **Citizen**: For residents reporting issues
   - **Service Provider**: For contractors/vendors
   - **Government Body**: For municipal departments

4. Fill in required information
5. Submit registration form
6. Login with your credentials

### Login

1. Enter your email and password
2. Click "Login"
3. You'll be redirected to your role-specific dashboard

## Citizen Guide

### Dashboard Overview

Your dashboard shows:
- **Total Complaints**: Number of complaints you've submitted
- **Status Breakdown**: Submitted, In Progress, Resolved counts
- **Complaint List**: Table of all your complaints

### Creating a Complaint

#### Step 1: Basic Information
- **Title**: Brief description (e.g., "Pothole on Oak Street")
- **Description**: Detailed explanation of the issue
- **Type**:
  - *Public*: Issues affecting community (roads, street lights, etc.)
  - *Personal*: Individual services needed (plumbing, electrical, etc.)
- **Category**: Select appropriate category
- **Severity**: Rate from 1 (minor) to 5 (critical)

#### Step 2: Location Details
- Enter the address/location of the issue
- Add nearby landmarks for easier identification
- GPS coordinates auto-detected if available

#### Step 3: Upload Images
- Upload photos of the issue
- Clear images help faster resolution
- Multiple images can be added

#### Step 4: Review & Submit
- Verify all information
- Click "Submit Complaint"
- You'll receive a confirmation notification

### Tracking Complaint Status

Status flow for **Public Complaints**:
1. **Submitted** → Received by system
2. **Under Review** → Being evaluated
3. **Assigned** → Sent to relevant department
4. **In Progress** → Work has started
5. **Resolved** → Issue fixed
6. **Closed** → Completed and verified

Status flow for **Personal Complaints**:
1. **Submitted** → Received by system
2. **Assigned** → Vendor assigned
3. **Pending Approval** → Estimate provided, awaiting approval
4. **In Progress** → Work started
5. **Resolved** → Work completed
6. **Closed** → Rated and closed

### Rating Service

After resolution:
1. View your resolved complaint
2. Click "Rate Service"
3. Give rating (1-5 stars)
4. Write optional review
5. Submit rating

## Vendor Guide

### Registration as Vendor

1. Select "Service Provider" during registration
2. Provide business details:
   - Business name
   - Service category
   - Years of experience
   - License number
   - Base service charge
   - Service radius (km)

3. Submit for verification
4. Admin will verify your license
5. Once verified, you can start accepting jobs

### Dashboard

Shows:
- **Available Jobs**: Nearby complaints in your category
- **Assigned Tasks**: Jobs you've accepted
- **Pending Estimates**: Quotes awaiting approval
- **Completed Jobs**: Successfully finished work
- **Your Rating**: Average customer rating

### Accepting Jobs

1. Browse "Available Complaints"
2. View complaint details
3. Click "Accept Job"
4. Job moves to "My Tasks"

### Providing Estimates

For personal complaints:
1. View assigned complaint
2. Assess the work required
3. Click "Submit Estimate"
4. Enter:
   - Labor cost
   - Material cost
   - Total cost
   - Detailed description
5. Submit for citizen approval
6. Wait for approval before starting work

### Completing Jobs

1. Start work → Mark as "In Progress"
2. Complete the work
3. Click "Mark Complete"
4. Add completion notes
5. Job marked as "Resolved"
6. Citizen can rate your service

### Managing Availability

Toggle availability:
- Go to Profile
- Update "Available" status
- Update location coordinates

## Government Guide

### Dashboard

Shows:
- **Assigned Complaints**: Issues assigned to your department
- **In Progress**: Active work items
- **Resolved**: Completed issues
- **Overdue**: Past SLA deadline

### Managing Complaints

#### Review Assigned Complaints
1. View list of complaints
2. Filter by status, priority, category
3. Click to view details

#### Assign to Team
1. Open complaint details
2. Click "Assign to Team"
3. Select appropriate team/worker
4. Add assignment notes
5. Status changes to "Assigned"

#### Start Work
1. When team begins work
2. Click "Start Work"
3. Status changes to "In Progress"
4. Add progress notes if needed

#### Resolve Complaint
1. After work is complete
2. Click "Resolve"
3. Add resolution notes describing work done
4. Submit
5. Citizen can rate the service

#### Reject Complaint
If complaint is not in your jurisdiction:
1. Click "Reject"
2. Provide clear reason
3. Admin will reassign

### Statistics

View department performance:
- Complaints by category
- Resolution times
- Team productivity
- Overdue items

## Common Tasks

### For All Users

#### Update Profile
1. Go to Profile page
2. Edit information
3. Save changes

#### Change Password
1. Go to Security settings
2. Enter current password
3. Enter new password
4. Confirm new password
5. Save

#### View Notifications
- Click notification bell icon
- View all notifications
- Mark as read

### Troubleshooting

#### Can't Login
- Verify email and password
- Check caps lock
- Reset password if forgotten
- Contact admin if account is deactivated

#### Complaint Not Routing
- Check if location is provided
- Verify category is selected
- Contact admin for manual assignment

#### Image Upload Fails
- Check file size (max 16MB)
- Ensure image format is supported (JPG, PNG, GIF)
- Try smaller file size

#### Not Receiving Notifications
- Check notification settings
- Verify email is correct
- Check spam folder

## FAQ

**Q: How long does complaint resolution take?**
A: Depends on priority:
- Critical: 1 day
- High: 3 days
- Medium: 7 days
- Low: 14 days

**Q: Can I edit my complaint after submission?**
A: Yes, but only while status is "Submitted" or "Under Review"

**Q: How are vendors selected?**
A: Based on proximity, category match, availability, and ratings

**Q: What if no vendor accepts my complaint?**
A: Admin will manually assign after 24 hours

**Q: Can I track complaint history?**
A: Yes, each complaint shows complete status history

**Q: How do I contact support?**
A: Use in-app messaging or contact admin
